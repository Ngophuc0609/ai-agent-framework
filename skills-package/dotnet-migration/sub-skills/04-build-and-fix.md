# Sub-skill: 04 Build and Fix

## Purpose
Ensure the ported code compiles, tests pass, and documentation is finalized.

## Instructions
1. Propose terminal commands (`dotnet build` and `dotnet test`) to verify the code.
2. If there are compilation errors, fix them iteratively.
3. If libraries are missing, propose `dotnet add package` commands to upgrade/install them.
4. Ensure all tests from Phase 3 pass successfully.
5. Finalize the `library-report.md` and `deferred-issues.template.md` and present the final status to the user.
