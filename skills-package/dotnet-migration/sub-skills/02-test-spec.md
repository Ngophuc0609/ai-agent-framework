# Sub-skill: 02 Test Spec Generation

## Purpose
Translate the baseline into concrete testing specifications before coding.

## Instructions
1. Load the completed baseline contract from Step 1.
2. Generate a Test Spec document mapping out the Unit Tests required for:
   - Every API Endpoint (I/O, Status Codes)
   - Every Service Logic (Business rules)
   - Background Jobs & Realtime events
3. The spec must define the input mocks and expected assertions.
4. Present the Test Spec to the user for approval.
