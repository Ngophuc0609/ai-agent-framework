# Writing Vietnamese Commit Message

## Vietnamese User Summary

Skill này viết commit message tiếng Việt dựa trên diff, test và tài liệu đã cập nhật.

## Workflow

1. Read git diff summary.
2. Read test and verification results.
3. Identify user-visible behavior.
4. Mention tests, API verification, and documentation updates when present.
5. Produce a concise Vietnamese commit message.

## Suggested Content

Include relevant lines such as:

- `Đã bổ sung test ...`
- `Đã xác minh bằng curl/Postman ...`
- `Đã cập nhật tài liệu API ...`
- `Đã cập nhật tài liệu debug/database ...`
